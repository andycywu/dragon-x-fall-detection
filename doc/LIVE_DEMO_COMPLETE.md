## 🚀 快速啟動指南

### 方法1: 使用啟動器 (推薦)
```bash
python demo_launcher.py
```

### 方法2: 直接啟動Streamlit
```bash
python demo_launcher.py --mode streamlit
```

### 方法3: 啟動普通Web Demo
```bash
python demo_launcher.py --mode web
```

### 方法4: 選擇啟動模式
```bash
python demo_launcher.py --mode both
```
